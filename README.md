# Iris Flower Classification Project 🌸

This project uses machine learning to classify iris flowers into three distinct species (*Setosa*, *Versicolor*, and *Virginica*) based on their sepal and petal measurements. It is built and executed entirely within Google Colab.

[![Open In Colab](https://google.com)](https://google.com)

## 📌 Project Overview
* **Dataset:** The classic Iris Dataset (4 features: Sepal Length, Sepal Width, Petal Length, Petal Width).
* **Goal:** Train a classification model to accurately predict flower species.
* **Libraries Used:** Python, Pandas, NumPy, Scikit-Learn.

## ⚙️ Workflow
1. **Data Loading:** Fetching the dataset natively from Scikit-Learn.
2. **Preprocessing:** Splitting the data into training and testing datasets.
3. **Model Training:** Training a classification algorithm on the training features.
4. **Evaluation:** Scoring the model's predictive performance on the test data.

## 🚀 How to Run the Project
1. Click the **Open in Colab** badge at the top of this file.
2. Once the notebook opens in Google Colab, click **Runtime** in the top menu.
3. Select **Run all** to execute all code cells sequentially.
4. The dataset is loaded automatically, so no manual file uploads are required.
